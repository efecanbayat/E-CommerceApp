# EcommerceApp


## About this project

This project is a application about ecommerce.

## Used Technologies

* MVVM Architecture
* Firebase
* Retrofit, okHttp
* Dagger Hilt
* Kotlin Coroutines
* ViewModel - LiveData
* Gson
* Lottie Animations
* RecyclerView
* Fragments
* Navigation Component

## Project Structure

* Splash Screen
* Product List Screen
* Product Detail Screen
* Product Add Screen
* Product Update Screen(Bottom Sheet)
